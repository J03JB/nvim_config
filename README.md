## Vim Tips & Tricks

### normal mode

`Ctrl+y` - to scroll screen up one line (without moving cursor)  
`Ctrl+e` - to scroll screen down one line (without moving cursor)  
`g;` - jump to last edit made  
`*` - to look for word under cursor  
`%` - to jump to  matching bracket  

### Insert mode

`Ctrl+t` - indent line  
`Ctrl+d` - outdent line
