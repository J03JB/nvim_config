# Config for Neovim 0.8


**Install Packer**:
``` bash
git clone --depth 1 https://github.com/wbthomason/packer.nvim\
 ~/.local/share/nvim/site/pack/packer/start/packer.nvim

```

## Plugins:
- LSPZero (autocompletion, lsp config)
- null-ls (formatting)
- NvimTree
- Tree sitter
- gitsigns
- lualine
- telescope
- truezen
- auto pairs
- bufferline
- indent blankline
